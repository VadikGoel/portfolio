## 🌌 Overview

An interactive monochrome portfolio showcasing projects, AI interests, game development progress and experiments.

## ✨ Features

- ⭐ Animated starfield background
- ✨ Mouse-follow glow
- 📱 Responsive design
- 🤖 AI Playground
- 🎮 Favorite games section
- 🧩 Smooth reveal animations
- 🗂 Project showcase

## 🛠 Tech Stack

```text
HTML5
CSS3
JavaScript
Intersection Observer API
```

## 🚀 Run Locally

```bash
git clone https://github.com/VadikGoel/portfolio.git
cd portfolio
```

Open `index.html`
